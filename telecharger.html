Toutes les options

Trois façons d'obtenir CORELESS, toutes hébergées sur les releases GitHub.

Recommandé

Installeur (x64)

Ajoute CORELESS au menu Démarrer, crée un raccourci bureau, gère la désinstallation proprement.

download Setup.exe
Sans installation

Portable (x64)

Un seul exécutable, à lancer directement. Rien n'est écrit dans le système, aucune désinstallation nécessaire.

bolt CORELESS.exe
Expérimental

Portable (arm64)

Pour Windows sur ARM. Le pilote capteurs reste x64 — préfère la version x64 (émulée) pour toutes les fonctions.

bolt CORELESS.exe

Prérequis & notes

SystèmeWindows 10 ou 11, 64 bits
DroitsAdministrateur au lancement (lecture températures/tensions)
DépendancesAucune — build self-contained, .NET inclus
Taille~70 Mo (installeur ou portable)
LicenceMIT — gratuit et open-source
L'exécutable n'est pas signé avec un certificat payant : Windows SmartScreen peut afficher « Éditeur inconnu ». C'est normal pour un projet indépendant non commercial — clique « Informations complémentaires » puis « Exécuter quand même ». Le code source complet est public sur GitHub.
